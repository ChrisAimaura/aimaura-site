The product

A new category of input device.

For years, players have been forced to choose: the precision and rapid aiming of a mouse, or the comfort and familiarity of a controller. Aimaura ends that compromise. Two ergonomically-shaped shells — one for each hand — combine high-resolution mouse tracking with analogue thumb-stick navigation, programmable side buttons, modular triggers and familiar ABXY inputs.

  • High-resolution optical mouse sensor on the right shell
  • Analogue joystick & D-Pad on the left shell
  • Programmable side buttons, bumpers and triggers on both
  • Wireless RF / BLE with USB-C charging on each unit
What's inside

Ten claims. One controller.

Every input is engineered for independent activation while the device is held ergonomically in a single, one-handed grip — left and right shells working as one.

Analogue Joystick

Multi-directional input via angular displacement of the control stick — driving on-screen movement and look from your left thumb.

Directional Pad

A familiar four-way D-Pad on the left shell registers directional commands for navigating menus and providing precise input during gameplay.

ABXY Cluster

The right shell carries a familiar ABXY button cluster, configured for actuation by the right thumb to execute user-assigned actions.

Mouse Tracking Sensor

A high-resolution optical sensor sits beneath the right shell, detecting relative movement across an underlying surface for cursor-grade precision.

Programmable Side Buttons

Three programmable buttons on the inside lateral surface of each shell — assignable to any user-selectable function for true input remapping.

Bumper & Trigger

Each shell carries a bumper and an analogue trigger along the upper edge — variable input levels for nuanced control under your index or middle finger.

USB-C Charging

USB-C on each shell supplies power to the internal battery and supports wired data — wireless when you want it, wired when you need it.

Wireless RF / BLE

Each unit transmits over RF / BLE to a paired receiver, where aggregation logic combines both shells into a single USB HID output for the host PC or console.

One-Handed Ergonomics

Every input is positioned for activation while each shell is held in a single, one-handed grip — no compromise on comfort, no sacrificed reach.

Industrial design

Sculpted for both hands.

A weighted base, a sculpted thumb crown, and a recessed trigger well. Each shell stands ready on the desk and tracks the moment you pick it up.

System architecture

Two shells, one HID device.

Each shell carries its own micro-controller, battery, and RF / BLE transmitter. Their signals are combined by aggregation logic in a paired receiver and presented to the host PC or console as a single USB HID output — so the device behaves as one to the operating system, while feeling like two natural extensions of your hands.

  • Independent MCUs per shell with on-board storage for profiles
  • Dual RF / BLE receiver with aggregation logic
  • USB HID output — no proprietary drivers, no custom emulator
